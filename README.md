# Meteor 프로젝트 생성 및 패키지 추가
```
meteor create [projectName] --blaze --release 2.6
npm install # npm 패키지 설치
meteor add autopublish # 개발용 서버 데이터베이스 자동 연동 패키지
meteor add insecure # 개발용 서버 데이터베이스 권한 획득
meteor add kadira:flow-router # 라우터 모듈 설치
meteor add kadira:blaze-layout # 라우터 레이아웃 모듈 설치
meteor # 프로젝트 실행. npm start 도 동일 효과
```
